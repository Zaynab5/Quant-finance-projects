# Monte Carlo Simulation for Option Pricing

## Overview
This project implements a Monte Carlo simulation to estimate the price of a European call option. The model simulates stock price paths using Geometric Brownian Motion under stochastic assumptions.

## Objective
To apply numerical methods for derivative pricing and compare simulated results with theoretical models such as Black-Scholes.

## Methodology
- Simulated 10,000+ stock price paths
- Used Geometric Brownian Motion to model price dynamics
- Calculated option payoff at maturity
- Discounted expected payoff to present value

## Results
The simulation provides an estimated option price and visualizes stock price paths, demonstrating convergence toward theoretical expectations.
